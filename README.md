## Quick Start

1. Download the latest release
2. Ensure you have Python installed
3. Run the application
4. Toggle "Prank Mode" to enable
5. Adjust probability slider to your preference
6. Continue your normal computer activities

## Requirements
- Python 3.x
- tkinter
- pygame
- PIL (Pillow)
- pynput
- pyautogui
